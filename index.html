HR Safety Campus - Sistem Manajemen Tenaga Kerja & Klaim Transportasi

Presentasi Sidang Magang - PT. Pratama Nusantara Sakti

Oleh: Nima Tamren

Tampilan Antarmuka Utama

Arsitektur Dashboard berbasis kartu (Grid System) untuk akses cepat ke setiap modul utama.

Manajemen Pengguna & RBAC

  • Administrator: Akses penuh via Gate::before.
  • User Biasa: Izin akses menu ditentukan secara granular (Permissions Array).

Alur A: Registrasi Tenaga Kerja

Fitur Unggulan:

  • Auto-ID (YYYYXXXX).
  • Validasi NIK 16 Digit.
  • Integrasi API Wilayah (IndoRegion).
  • Status Otomatis: Pending jika data belum lengkap (misal: Fingerprint kosong).

Kontrol Kualitas: Approval

Hanya data berstatus "Lengkap" yang dapat di-approve massal untuk menjadi karyawan Aktif.

Alur B: Klaim Transportasi

Logika Bisnis Berlapis:

  • Cek masa kerja (≥60 hari untuk IN, ≥90 hari untuk OUT).
  • Validasi jeda klaim 365 hari.
  • Nominal otomatis berdasarkan Tabel Tarif.

Manajemen Tarif & Audit Trail

Setiap update tarif akan mengarsipkan data lama secara otomatis (Rate Periods) dan wajib melampirkan dokumen PDF pengesahan.

Alur C: Surat Izin Keluar (Permit)

Otomasi nomor surat & perubahan status Non-Aktif jika memilih jenis "Keluar Selamanya".

Master Data & History

Manajemen data aktif, fitur Re-Hire (Soft Delete), dan cetak ID Card per individu atau rombongan.

Visualisasi & Analitik

Ringkasan populasi, status fingerprint, dan demografi menggunakan Chart.js.

Tantangan & Solusi Teknis

✅ Migrasi Data: Deduplikasi NIK pada ~4.000 data MS Access.

✅ Konsistensi: Penomoran berbasis max(no_id) untuk menghindari duplikasi pasca penghapusan data.

✅ Integritas: Penggunaan DB::transaction pada proses pengarsipan tarif dan klaim massal.
